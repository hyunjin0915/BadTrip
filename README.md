## 프로젝트 소개 | Project Introduction

### 2D 픽셀 그래픽 스토리텔링형 시리어스 게임

청소년 마약 범죄의 심각성, 위험성을 알리고

마약 중독에 대한 경각심을 고취시키는 것이 목표인 **스토리텔링형 시리어스 게임** 콘텐츠
